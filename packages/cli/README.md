# @wishcraft/cli

WishCraft CLI for creator template projects.

## Commands

- `wishcraft init`
- `wishcraft validate`
- `wishcraft build`
- `wishcraft login`
- `wishcraft submit`

## Template project shape

`wishcraft init` scaffolds:

- `src/index.tsx`
- `config.json`
- `preview.png`
- `package.json`
- `tsconfig.json`
- `README.md`

## Development

```bash
npm run build
npm run typecheck
```
